Consecutive Number Checker

Input a number in the box below and this application will calculate whether an amount of consecutive numbers based on that number add up to a multiple of that original number

e.g if the input is 3. The consecutive numbers will be 1, 2, 3. These numbers added together equals 6, and 6 divided by 3 (the original number) equals 2 and is a integer (whole number) and therefore a multiple of 3 so the result returns true. If the end result is not an integer it returns false.

Further explanation

If using 3 as the starting number

An array of consecutive numbers could be 1, 2, 3 or 7, 8, 9 or 84, 85, 86

When these numbers are added up are they a multiple of the original number of 3?

This application will work this out // futher explanation within the javaScript file

The reason this application only needs the starting number as it will mimic an array from the original number. This 'pretend array' only contains differences in value from each number to the starting number in the array.
